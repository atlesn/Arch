# dwm 

dwm 6.2

This build of dwm uses the following two fonts:

* extra/ttf-hack
* aur/nerd-fonts-roboto-mono

Please note that Roboto Mono Nerd Font must be installed manually.

The following patches are installed:

* attachbelow
* autostart
* rotatestack
* ru_gaps
* status2d
* statuspadding
* switchtotag
* titlecolor
